## 使用示例

### 示例 1: SSH Key 方式

```bash
# 使用默认 SSH Key
python cli.py --private-repo --ssh-key ~/.ssh/id_rsa

# 使用自定义 SSH Key
python cli.py --private-repo --ssh-key /path/to/custom/key --output report.md
```

### 示例 2: Token 方式

```bash
python cli.py --private-repo \
  --token "ghp_xxxxxxxxxxxxxxxxxxxx" \
  --remote-url "https://github.com/company/private-project.git" \
  --author "张三" \
  --output weekly_report.md
```

### 示例 3: 配置文件方式

```bash
# 创建配置文件
cat > config_private.json << 'EOF'
{
  "private_repo": true,
  "git_auth": {
    "auth_type": "token",
    "token": "your_token_here",
    "remote_url": "https://github.com/username/repo.git"
  },
  "author_name": "张三",
  "highlights": ["完成核心功能"]
}
EOF

# 使用配置文件
python cli.py --config config_private.json --output weekly_report.md
```

### 示例 4: Python 代码调用

```python
from weekly_report_generator import ReportGenerator, ReportConfig, GitAuthConfig

# 配置 Git 认证
git_auth = GitAuthConfig(
    auth_type="token",
    token="ghp_xxxxxxxxxxxxxxxxxxxx",
    remote_url="https://github.com/username/private-repo.git"
)

# 创建配置
config = ReportConfig(
    report_type="weekly",
    private_repo=True,
    git_auth=git_auth,
    author_name="张三"
)

# 生成报告
generator = ReportGenerator(config)
report = generator.generate()
```

## 安全提示

### ⚠️ 重要安全建议

1. **不要提交敏感信息**
   ```bash
   # 将以下文件添加到 .gitignore
   config_private.json
   *.secret.json
   ```

2. **设置文件权限**
   ```bash
   chmod 600 config_private.json
   chmod 600 ~/.ssh/id_rsa
   ```

3. **使用环境变量**
   ```bash
   export GIT_TOKEN="your_token"
   python cli.py --private-repo --token $GIT_TOKEN
   ```

4. **定期更换凭证**
   - Token 建议 30-90 天更换一次
   - 设置 Token 过期时间

## 常见问题

### Q: 如何获取 GitHub Token？

A: 
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 选择权限：至少勾选 `repo`
4. 生成并复制 Token

### Q: SSH Key 放在哪里？

A: 
- 默认位置：`~/.ssh/id_rsa` 或 `~/.ssh/id_ed25519`
- 自定义位置：使用 `--ssh-key /path/to/key` 指定

### Q: Token 安全吗？

A: 
- Token 本身是安全的，但需要妥善保管
- 建议设置过期时间
- 不要提交到 Git 仓库
- 使用环境变量或加密存储

### Q: 支持哪些 Git 平台？

A: 
- ✅ GitHub
- ✅ GitLab
- ✅ Gitee
- ✅ Bitbucket
- ✅ 任何支持 SSH 或 Token 的 Git 平台
