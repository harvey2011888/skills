## 🎯 参赛信息

### 参赛赛道
**Skill 创作赛道**

### 作品名称
**周报/月报自动生成器**

### 一句话介绍
从 Git 提交记录、任务系统、文档变更中自动提取工作成果，一键生成结构化周报/月报的 Smart Assistant。

### 核心价值
- **节省时间**：从 60 分钟缩短到 5 分钟
- **内容完整**：自动收集，不会遗漏工作内容
- **数据支撑**：基于真实的 Git 提交和任务数据
- **格式统一**：标准化模板，提升专业度

## 🏗 技术架构

```
┌─────────────────────────────────────────────┐
│          周报/月报自动生成器                │
├─────────────────────────────────────────────┤
│  输入层                                     │
│  - Git 仓库                                 │
│  - 任务文件 (JSON/Markdown)                 │

├─────────────────────────────────────────────┤
│  处理层                                     │
│  - GitExtractor: 提取 Git 提交记录          │

├─────────────────────────────────────────────┤
│  生成层                                     │
│  - ReportGenerator: 报告生成引擎            │
│  - 模板系统                                 │
│  - 格式化输出                               │
├─────────────────────────────────────────────┤
│  输出层                                     │
│  - Markdown 格式报告                        │
│  - 支持扩展 HTML/PDF                        │
└─────────────────────────────────────────────┘
```

## 🚀 使用方式

### 1. 命令行工具
```bash
python cli.py --author "张三" --team "开发组" --output weekly_report.md
```

### 2. Python 代码
```python
from weekly_report_generator import ReportGenerator, ReportConfig
config = ReportConfig(author_name="张三", highlights=["完成核心功能"])
generator = ReportGenerator(config)
report = generator.generate()
```
