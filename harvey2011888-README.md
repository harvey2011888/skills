#### 🧹 Windows C 盘清理 🆕
智能扫描和清理 Windows C 盘，先列出可清理项目供选择，确认后执行并生成报告。**支持 Win7/8/8.1/10/11。**
**核心特性：**
- 🔍 智能扫描 — 覆盖 20+ 种清理项（临时文件、更新缓存、浏览器缓存、系统转储等）
- 🛡️ 安全清理 — 内置白名单保护，自动跳过 System32 等关键系统目录
- 📋 交互式选择 — 扫描后列出清单，用户选择后才执行
- 📊 详细报告 — 清理前后对比、成功/失败明细
- 🌐 全版本兼容 — Win7 / Win8 / Win8.1 / Win10 / Win11
- 📦 零依赖 — 仅使用 Python 标准库
