# Windows C 盘清理工具

## 技能说明

这是一个智能 Windows C 盘清理助手，可以安全地扫描和清理系统 C 盘中的临时文件、缓存、日志等，释放宝贵的磁盘空间。

## 核心功能

- **智能扫描** - 自动扫描多种常见的占用空间项目
- **安全清理** - 内置白名单保护，跳过系统关键文件
- **权限管理** - 智能检测管理员权限需求
- **分类清理** - 支持选择性清理不同类型的文件
- **详细报告** - 提供清晰的扫描和清理报告
- **零依赖** - 仅使用 Python 标准库，开箱即用

## 清理项目

| 清理项 | 说明 | 安全性 |
|--------|------|--------|
| 用户临时文件 | 应用程序的临时文件 | ✅ 安全 |
| Windows 临时文件 | Windows 系统临时文件 | ✅ 安全 |
| Windows 更新缓存 | 更新下载的缓存文件 | ✅ 安全 |
| 预读取文件 | 系统预读取文件 | ✅ 安全 |
| 回收站 | 已删除的文件 | ✅ 安全 |
| Windows 日志 | 系统日志文件 | ✅ 安全 |
| 错误报告 | Windows 错误报告文件 | ✅ 安全 |
| 浏览器缓存 | Chrome/Firefox/Edge 缓存 | ✅ 安全 |
| 下载文件夹 | 下载的文件 | ⚠️ 需确认 |

## 使用场景

- C 盘空间不足，需要释放空间
- 磁盘空间告警，需要清理
- 定期维护系统，保持磁盘健康
- 清理浏览器缓存释放空间
- 清理 Windows 更新残留文件

### 方式三：Python 代码调用

```python
from windows_cleaner import CleanerScanner, CleanerExecutor

# 扫描
scanner = CleanerScanner()
scan_result = scanner.scan()

print(f"总占用空间: {scan_result.total_size} 字节")
print(f"文件总数: {scan_result.total_files}")

# 清理所有安全项目
executor = CleanerExecutor()
clean_result = executor.clean_items(
    [item for item in scan_result.items if item.is_safe],
    dry_run=False
)

print(f"已清理: {clean_result.cleaned_size} 字节")
print(f"清理文件数: {clean_result.cleaned_files}")
```

## 命令行参数

### 扫描命令

```bash
python cli.py --scan
```

扫描 C 盘可清理项目，输出详细的扫描报告。

**参数：**
- `--scan`: 执行扫描操作
- `--json`: 输出 JSON 格式（便于程序处理）

### 清理命令

```bash
# 清理所有安全项目
python cli.py --clean-all

# 清理指定项目
python cli.py --clean "项目1" "项目2"

# 模拟运行（不删除）
python cli.py --clean-all --dry-run

# 输出 JSON 格式
python cli.py --clean-all --json
```

**参数：**
- `--clean <项目名>`: 清理指定项目（可指定多个）
- `--clean-all`: 清理所有安全项目
- `--dry-run`: 模拟运行，不实际删除
- `--json`: 输出 JSON 格式

## 安全机制

### 白名单保护
- 自动跳过 Windows 系统关键目录
- 保护系统启动文件和配置文件
- 避免删除正在使用的文件

### 权限管理
- 自动检测是否需要管理员权限
- 提示用户提升权限
- 部分清理项支持普通用户权限

### 用户确认
- 显示清理前的大小和文件数
- 支持选择性清理
- 提供详细清理报告

## 注意事项

1. **备份重要数据** - 清理前请确认没有重要文件在目标目录
2. **关闭应用程序** - 建议关闭正在使用浏览器的应用程序
3. **管理员权限** - 部分清理项需要管理员权限
4. **下载文件夹** - 清理前请检查下载文件夹是否有重要文件
5. **谨慎操作** - 不建议清理系统关键目录下的文件

## 常见问题

### Q: 清理会删除重要文件吗？
A: 不会。工具有完善的白名单保护机制，会自动跳过系统关键文件和正在使用的文件。

### Q: 需要管理员权限吗？
A: 部分清理项需要管理员权限。工具会自动检测并在需要时提示。

### Q: 可以只清理特定项目吗？
A: 可以。使用 `--clean` 参数指定要清理的项目名称。

### Q: 如何查看清理的详细信息？
A: 使用 `--json` 参数输出 JSON 格式，便于程序处理。

### Q: 清理后会影响系统运行吗？
A: 不会。清理的都是临时文件、缓存等不影响系统正常运行的文件。
