# Windows C 盘清理工具 v2.0

## 技能说明

这是一个智能的 Windows C 盘清理助手。当用户请求清理 C 盘时，你必须严格按照以下交互流程执行，不可跳过任何步骤：

## 交互流程

### 步骤 1: 扫描

首先运行扫描命令，获取所有可清理项目：

```bash
python src/cli.py --scan --json
```

解析返回的 JSON 数据，获取系统版本、磁盘信息、各清理项的名称、路径、占用大小、文件数、安全性、是否需要管理员权限等。

**重要**: 必须在终端中实际执行此命令，不得虚构或模拟结果。

### 步骤 2: 列出清理项，让用户选择

将扫描结果以清晰的表格形式展示给用户，包含以下信息：
- 编号、清理项目名称、占用空间、文件数量
- 每条的安全性标识（✅ 安全 / ⚠️ 需确认）
- 是否需要管理员权限（🔒 需管理员 / 🔓 无需管理员）

然后使用 `AskUserQuestion` 工具询问用户要清理哪些项目，提供以下选项：

**选项示例：**
- "全部安全项目（推荐）" - 清理所有标记为安全的项目
- "临时文件 + 系统缓存" - 清理临时文件、系统临时文件、预读取文件
- "临时文件 + 浏览器缓存" - 清理临时文件和所有浏览器缓存
- "仅临时文件" - 只清理用户临时文件
- "自定义选择" - 用户自行输入编号

如果用户选择"自定义选择"，则让用户输入要清理的项目编号（如 1,3,5,7）。

**所有需要管理员权限的清理项，请明确提醒用户**。

### 步骤 3: 确认后执行清理

用户选择完毕后，再次确认是否执行清理。确认后，运行以下命令执行清理：

```bash
python src/cli.py --clean "项目名1" "项目名2" ... --json
```

或者使用编号方式（Python 调用）：

```python
from windows_cleaner import clean_by_indices
result = clean_by_indices([1, 3, 5, 7])
```

**重要**: 必须在终端中实际执行清理命令，不得虚构结果。

### 步骤 4: 生成清理报告

清理完成后，将结果整理成清晰的报告展示给用户：

```
==============================================================
  Windows C 盘清理 - 清理报告
==============================================================
  清理时间:      2026-05-21 15:30:00
  清理前占用:    5.23 GB (15,432 个文件)
  已清理大小:    2.15 GB
  已清理文件:    8,234
  成功清理:      5 项

  已清理项目:
  ------------------------------------------------------------
    1. 临时文件
    2. 系统临时文件
    3. Chrome 缓存
    4. Edge 缓存
    5. Windows 更新缓存

  清理失败项: (如有)
  ------------------------------------------------------------
    1. 系统日志
       路径: C:\Windows\Logs
       原因: 需要管理员权限，请以管理员身份运行

==============================================================
```

报告必须包含：
- 清理时间
- 清理前的总占用大小和文件数
- 实际清理的大小和文件数
- 成功清理的项目列表
- 失败的项目及原因（如有）

如果发现有需要管理员权限但当前没有的失败项，提示用户以管理员身份重新运行。

|--------|------|--------|------|----------|

| 系统临时文件 | `%WINDIR%\Temp` | ✅ 安全 | 🔒 需管理员 | 全部 |
| Windows 更新缓存 | `%WINDIR%\SoftwareDistribution\Download` | ✅ 安全 | 🔒 需管理员 | 全部 |

| 回收站 | `$Recycle.Bin` | ✅ 安全 | 🔒 需管理员 | 全部 |
| 缩略图缓存 | `%LOCALAPPDATA%\Microsoft\Windows\Explorer` | ✅ 安全 | 🔓 无需 | Win10/11 |
| 系统日志 | `%WINDIR%\Logs` | ✅ 安全 | 🔒 需管理员 | 全部 |
| 安装日志 | `%WINDIR%\Panther` | ✅ 安全 | 🔒 需管理员 | 全部 |
| 错误报告 | `%ProgramData%\Microsoft\Windows\WER` | ✅ 安全 | 🔒 需管理员 | 全部 |
| DirectX 着色器缓存 | `%LOCALAPPDATA%\D3DSCache` | ✅ 安全 | 🔓 无需 | Win10/11 |
| 旧版 Windows | `C:\Windows.old` | ✅ 安全 | 🔒 需管理员 | 全部 |
| 系统内存转储 | `%WINDIR%\MEMORY.DMP` | ✅ 安全 | 🔒 需管理员 | 全部 |
| 小型内存转储 | `%WINDIR%\Minidump` | ✅ 安全 | 🔒 需管理员 | 全部 |
| 浏览器缓存 | Chrome/Edge/Firefox/Opera/Brave 等 | ✅ 安全 | 🔓 无需 | 全部 |
| 下载文件夹 | `%USERPROFILE%\Downloads` | ⚠️ 需确认 | 🔓 无需 | 全部 |

**浏览器缓存覆盖**: Chrome、Edge、Firefox、Opera、Brave、Vivaldi、Chromium、360安全浏览器、360极速浏览器、QQ浏览器、搜狗浏览器、微信内置浏览器。

## 命令行使用参考

### 扫描
```bash
python src/cli.py --scan              # 完整扫描报告
python src/cli.py --scan --json       # JSON 格式（推荐编程调用）
python src/cli.py --list              # 简洁表格格式
```

### 清理
```bash
python src/cli.py --clean-all                     # 清理所有安全项目
python src/cli.py --clean-all --dry-run           # 模拟运行
python src/cli.py --clean "临时文件" "Chrome 缓存"  # 清理指定项目
python src/cli.py --interactive                   # 交互式选择模式
```

## 安全机制

- **白名单保护**: 自动跳过 System32、SysWOW64、Boot 等关键系统目录
- **权限检测**: 区分需要/不需要管理员权限的清理项
- **用户确认**: 所有清理操作需用户确认后执行
- **安全删除**: 遍历删除前逐文件安全检查

## 系统要求

## 常见问题

### Q: 清理会删除重要文件吗？
A: 不会。工具有多层安全保护，自动跳过系统关键文件和正在使用的文件。

### Q: 需要管理员权限吗？
A: 部分清理项需要，工具会明确标注。建议以管理员身份运行以清理更多项目。

### Q: 清理后会影响系统运行吗？
A: 不会。清理的都是临时文件、缓存、更新残留等不影响系统正常运行的文件。

### Q: 支持哪些 Windows 版本？
A: 支持 Windows 7/8/8.1/10/11 全部主流版本。
