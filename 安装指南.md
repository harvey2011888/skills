## ⚠️ 常见问题

### Q: 为什么之前上传失败？
A: 因为原项目缺少 `SKILL.md` 文件。SOLO 要求所有 Skill 包必须包含这个文件来定义技能的元数据。

### Q: SKILL.md 是什么？
A: SKILL.md 是 SOLO Skill 的标准描述文件，包含：
- YAML 格式的元数据（name, description, version 等）
- Markdown 格式的详细说明
- 使用示例和配置说明

### Q: 如何验证 SKILL.md 是否正确？
A: 正确的 SKILL.md 应该：
1. 以 `---` 开头和结尾的 YAML 元数据块
2. 包含必需的字段：name, description, version
3. 包含 Markdown 格式的详细说明

### Q: 安装后无法使用怎么办？
A: 检查：
1. Python 3.7+ 是否安装：`python --version`
2. Git 是否安装：`git --version`
3. 重启 SOLO 应用
